# Caesar Cipher Encryption & Decryption Tool

## 📌 Internship Task
This project was developed as part of my Cyber Security Internship at **Prodigy Infotech**.

## 📖 Project Description
The Caesar Cipher is one of the simplest and most widely known encryption techniques.  
In this project, a Python program is created to perform **encryption and decryption** of text using the Caesar Cipher algorithm.

The program shifts each letter of the plaintext by a specified number of positions in the alphabet to generate encrypted text. It can also reverse the process to decrypt the message.

## 🚀 Features
- Encrypt text messages
- Decrypt encrypted messages
- User-defined shift value
- Handles both uppercase and lowercase letters
- Keeps spaces and symbols unchanged
